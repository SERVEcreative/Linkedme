# Contributing to Professional Networking Platform

Thank you for your interest in contributing to this project!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/linkedme.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit your changes: `git commit -m 'Add some feature'`
6. Push to the branch: `git push origin feature/your-feature-name`
7. Open a Pull Request

## Development Setup

1. Ensure you have Java 17+ and Maven 3.8+ installed
2. Start infrastructure services: `docker-compose -f docker/docker-compose.yml up -d`
3. Run services individually or use your IDE

## Code Style

- Follow Java coding conventions
- Use meaningful variable and method names
- Add comments for complex logic
- Write unit tests for new features

## Commit Messages

- Use clear, descriptive commit messages
- Reference issue numbers if applicable
- Follow conventional commit format when possible

## Testing

- Write unit tests for all new features
- Ensure all tests pass before submitting PR
- Add integration tests for API endpoints

## Questions?

Feel free to open an issue for any questions or suggestions.
